# GitWorld Screensaver

A macOS screensaver that cycles through every world generated by the git_world community.

Every time someone generates a world on the site, your screensaver gets it. An infinite, ever-growing gallery of open source — rendered as cinematic 3D worlds.

## Quick Test (no build needed)

Open the HTML file directly in your browser to preview:

```bash
open screensaver/GitWorld.saver/Contents/Resources/screensaver.html
```

Or use it with [WebViewScreenSaver](https://github.com/nickytonline/web-view-screen-saver):
1. Install WebViewScreenSaver
2. Point it to the deployed URL: `https://git-world-m9aes.ondigitalocean.app/api/history`
3. Or to the local HTML file

## Build Native Screensaver

```bash
cd screensaver
./build.sh
```

This compiles the Swift screensaver bundle and installs it to `~/Library/Screen Savers/`.

Then: **System Settings → Screen Saver → GitWorld**

## Test Immediately

```bash
/System/Library/CoreServices/ScreenSaverEngine.app/Contents/MacOS/ScreenSaverEngine
```

## How It Works

1. Fetches all worlds from `git-world-m9aes.ondigitalocean.app/api/history`
2. Shuffles and cycles through them every 25 seconds
3. Crossfades between panoramas with smooth transitions
4. Genre-matched ambient music and particle colors
5. Re-fetches every 10 minutes for new community worlds
6. Works offline with cached worlds

## Uninstall

```bash
rm -rf ~/Library/Screen\ Savers/GitWorld.saver
```
